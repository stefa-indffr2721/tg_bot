Томашевская Дарья и Пестерева Виктория КН-202

QuizBot: Развертывание на Railway

Этот проект представляет собой Telegram-бота для викторин с вопросами из различных категорий, поддерживающего как одиночную игру, так и дуэли.

Предварительные требования:

·Учетная запись на GitHub.

·Учетная запись на Railway.

·Токен вашего бота, полученный от BotFather в Telegram.

Инструкция по развертыванию на Railway

Шаг 1: Подготовка проекта

Убедитесь, что ваш проект содержит следующие ключевые файлы:

·pom.xml - конфигурация Maven для Java-проекта.

·Dockerfile - инструкции для сборки контейнера.

·Файлы исходного кода (src/), настроенные на чтение переменных окружения для токена бота.

Шаг 2: Создание проекта на Railway

·Войдите на Railway.

·Нажмите "New Project" и выберите "Deploy from GitHub repo".

·Предоставьте доступ к нужному репозиторию и выберите его.

Railway автоматически обнаружит Dockerfile и начнет процесс сборки.

Шаг 3: Настройка переменных окружения

Для безопасной работы бота необходимо задать конфиденциальные данные как переменные окружения в Railway Dashboard:

·В панели вашего проекта на Railway перейдите на вкладку "Variables".

·Добавьте следующие переменные:

-BOT_TOKEN = ваш_секретный_токен_от_BotFather

-BOT_USERNAME = имя_вашего_бота

После добавления переменных Railway автоматически перезапустит приложение с новыми настройками.

Шаг 4: Деплой и проверка

После настройки переменных и успешной сборки ваш бот будет запущен.

В логах проекта (вкладка "Logs" в Railway) не должно ничего появиться.

Перейдите в Telegram и найдите своего бота по имени (имя_вашего_бота).
